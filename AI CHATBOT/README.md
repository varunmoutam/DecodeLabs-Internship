# AI Assistant - Rule-Based Chatbot

A professional rule-based chatbot built with Python Flask featuring a modern dark UI.

## Features

- **General Conversation**: Greetings, farewells, small talk
- **General Knowledge**:
  - Capital cities (20+ countries)
  - Planet facts
  - Science concepts
  - Historical events
  - Mathematical concepts
- **Basic Calculations**: +, -, *, /, ^ operations
- **Time & Date**: Current time and date
- **Jokes**: Light entertainment
