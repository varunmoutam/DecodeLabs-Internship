# 🤖 Rule-Based Chatbot using Python & Flask

A simple Rule-Based Chatbot built using Python, Flask, HTML, CSS, and JavaScript. This chatbot can handle general conversations, answer general knowledge questions, perform simple calculations, and provide fun interactive responses.

---
LIVE DEMO 

https://decodelabs-intenship.onrender.com

## 🚀 Features

- Greeting and farewell responses
- General conversation support
- Capital cities of countries
- Science facts and concepts
- History facts
- Mathematics facts
- Basic calculator operations
- Jokes and fun responses
- Motivational quotes
- Coin toss simulation
- Dice roll simulation
- India General Knowledge
- Iron Man facts
- Personalized Varun responses
- Flask-based web interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/rule-based-chatbot.git
cd rule-based-chatbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will run locally at:

```text
http://127.0.0.1:5000
```
